# ECU Emulator (RP2040)

## Überblick (Deutsch)
- Zweck: CAN/K-Line ECU-Emulator für Tests/Diagnose.
- Hardware: RP2040 Zero, OLED (I2C), optional WS2812 Status-LED.
- Build: Nutze die Artefakte im Verzeichnis `build/`.
- Nutzung: Lade die Binärdateien aus `build/` auf das Gerät (z.B. via UF2/Flash-Tool).
- Einschränkung: Dieses Repo veröffentlicht bewusst nur `build/` und diese README.

## Обзор (Русский)
- Назначение: эмулятор ЭБУ по CAN/K-Line для тестов/диагностики.
- Аппаратная база: RP2040 Zero, OLED (I2C), опционально адресный светодиод WS2812.
- Сборка: используйте артефакты в каталоге `build/`.
- Использование: заливайте бинарники из `build/` на устройство (UF2/Flash-Tool).
- Ограничение: в репозитории публикуются только `build/` и данный README.
